## RUN:

### Frontend:

- Go to client-side
- Fill the Environment Variables
- RUN on Terminal:

```shell script
npm i
npm run dev
```

### Backend:

- Go to server-side
- Fill the Environment Variables
- RUN on Terminal:

```shell script
npm i
node index.js
```
